# ⚙️ Etapas do Pipeline

## Checkout
Baixa o código fonte.

## Build
Compila o projeto Spring Boot.

## Tests
Executa testes unitários.

## Docker Build
Gera imagem da aplicação.

## Deploy
Executa container localmente.

## Health Check
Valida endpoint `/health`.

## Evidence
Gera JSON com evidência de execução.
